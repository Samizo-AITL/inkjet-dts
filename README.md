# Inkjet Quality vs Speed (Educational Model)

## 目的
インクジェット印刷における
- 画質（粒状性）
- 印刷速度
のトレードオフを理解する。

## モデルの考え方
- 解像度が上がる → 単位面積あたりの滴数が増える
- パス数が増える → 同じ場所に何度も吐出する
- 滴数が増える → 粒状性は改善するが速度は落ちる
